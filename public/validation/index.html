glTF Validator

Run the official Khronos glTF 2.0 validator and a glTF-Transform performance inspector — fully client-side. Optionally sign & pin reports on-chain.

◈
Khronos spec check
Validates glTF 2.0 file structure, accessors, materials and extensions against the official Khronos suite.
◆
Performance inspector
Surfaces what the spec check does not — texture weights, draw-call shape, and compression opportunities (Draco, Meshopt, KTX2).
◇
On-chain attestations
Sign reports with a wallet, pin to IPFS, and record on the Validation Registry so anyone can verify the audit.
CORS must be allowed by the host server.
