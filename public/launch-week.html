01.5 — What we shipped

The whole stack, in one repo.

three.ws is organized as four communicating layers, plus a backend and a contracts package. Every layer is in production at three.ws and ships as part of the open-source repo. Here's the inventory.

L4 Embed & Distribution
Drop-in 3D agents, anywhere on the web.

The <agent-3d> custom element loads lazily on intersection, supports five widget variants (turntable, animation gallery, talking agent, ERC-8004 passport card, hotspot tour), and emits Open Graph and oEmbed metadata so links unfurl with rich previews. A point-and-click Widget Studio generates copy-paste embed code at /studio. Versioned CDN bundles ship at /agent-3d/x.y.z/.

<agent-3d> 5 widget types Open Graph + oEmbed Widget Studio Versioned CDN postMessage API
L3 Identity & Persistence
On-chain identity. Signed action history.

Three Solidity contracts make up the ERC-8004 implementation: IdentityRegistry (ERC-721 agent tokens with delegated EIP-712 signers), ReputationRegistry (signed reviewer feedback, averaged on-chain), and ValidationRegistry (validator attestations for off-chain proofs). Every speak, remember, and skill-done event is signed and logged. Solana ships an analog using Metaplex Core NFTs plus signed SPL Memo attestations — same shape, no custom program required.

ERC-8004 ERC-721 tokens EIP-712 signing EIP-7710 delegation Foundry tested Solana SIWS + Metaplex IPFS pinning
L2 Agent Runtime
An LLM brain, with a body that feels.

The runtime drives a Claude tool-loop with up to eight iterations per turn and six built-in tools (wave, lookAt, play_clip, setExpression, speak, remember). The Empathy Layer is not a state machine — it's a continuous weighted blend of five emotions (celebration, concern, curiosity, empathy, patience), each with its own decay half-life, driving the avatar's morph targets and head pose every frame. Skills are self-contained bundles distributable over IPFS, Arweave, or HTTP. Memory is file-based with four storage modes (local, IPFS, encrypted-IPFS, none).

Claude tool-loop 8-iteration limit Empathy Layer Skill registry Web Speech + ElevenLabs YAML-front-matter memory
L1 Viewer
three.js r176, Khronos-grade glTF.

WebGL 2.0 rendering with full glTF 2.0 / GLB support: Draco geometry compression, KTX2 texture compression, Meshopt mesh optimization, HDR environment maps, PBR materials, skinned animation, morph targets, and embedded cameras. Validation is line-level via the Khronos gltf-validator. Drag-and-drop loading, OrbitControls with configurable auto-rotation, and a real-time dat.GUI for live tweaking.

three.js r176 Draco / KTX2 / Meshopt PBR + HDR glTF validator Drag-and-drop dat.GUI
B Backend
~153 serverless endpoints. ~3,600 lines of API.

Vercel functions on Neon Postgres, Cloudflare R2, and Upstash Redis. A full OAuth 2.1 server (RFC 6749 + PKCE, RFC 7591 dynamic registration, RFC 7009 revocation, RFC 7662 introspection, RFC 8414 discovery, RFC 9728 protected-resource discovery). An MCP server over HTTP — 759 lines, JSON-RPC 2.0, listed on the official Model Context Protocol Registry as io.github.nirholas/three.ws. Cron jobs for the ERC-8004 crawler, EIP-7710 delegation indexer, DCA strategy execution, and recurring on-chain subscriptions.

08 — Lessons

What worked, by the data

Ship daily. Name the partner. Of the 64 posts in this window, the highest-engagement ones were either (a) "we just shipped X, here's the link" or (b) "@PartnerName picked us up." Roadmaps and teasers do not exist in the top 20 by views. Every meaningful spike was a real shipped artifact or a real organization saying our name.

The flywheel is real. Apr 29 founder launch (135K views) → npm publish + MCP Registry + x402scan listing same day → Google Cloud announcement Apr 30 → Pump.fun skills + reverse-engineered SDK same window → Alibaba + CoinGecko May 2 → Solana Hackathon May 4 (42K views, 621 RTs) → Coinbase + Jupiter + Investing.com all stacked into May 5 (the 46K-impression peak day). One post earns the next.

Reply ratio > like ratio. 845 replies on 7,200 likes is an 11.7% reply rate — well into "live conversation" territory. The 1,010-member X community is the asset that ratio builds. Likes are sentiment; replies are people who showed up to talk.

Asymmetric posture is a signal. Following 11. Followed by 3,675. The numbers say: we are a publisher, not a follow-back farm. Verified followers (359, ~10%) skew toward power users — exactly the cohort that pays for, builds with, and amplifies what we ship next.

Cross-pollinated authors compound. The week's biggest individual post (135K views) came from @nichxbt, not @trythreews. The May 7 RT from @zhusu (50K views) was earned by the work, not the brand handle. Use both — founder voice for narrative, brand for ecosystem announcements.

Weekend doesn't sleep. The biggest single day (May 5) was a Sunday. The biggest single post (Solana Hackathon, 42K views, 621 RTs) hit on a Saturday. Crypto X is awake when most marketing teams aren't. Ship on the weekend.

Headline number for week one: 267,800 impressions, 25,300 engagements, 9.4% engagement rate, 3,675 followers, 1,010 community members, two npm packages, three live x402 endpoints, two cloud partnerships, four exchange listings, an MCP Registry slot, W3C membership, a Solana hackathon entry, and a published Vogue feature.
